Wine Quality Test

Build the Wine Quality ML Model using the Gradient Boosting Classifier to Classify the Wine Quality based on the Different Paramaters like, pH, Color, Acidity, sulphates ,... and saved the Model. Later used the saved model in the Flask app to predict and categorize Wine quality based on the user input into 3 differnt Categories Bad-Quality, Good-Quality and Best-Quality.
